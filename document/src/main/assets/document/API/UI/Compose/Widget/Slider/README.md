## 概况

[Slider](/API/UI/Compose/Widget/Slider/README.md) 可组合项用于显示滑块,允许用户从一系列值中进行选择。 滑块沿条反映一系列值，用户可以从中选择单个值。
它们是调整音量、亮度或应用图像滤镜等设置的理想选择。继承自 [Composable](/API/UI/Compose/Widget/Composable/README.md)
