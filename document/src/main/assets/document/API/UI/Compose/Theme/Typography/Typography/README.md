## 概况

[Typography](/API/UI/Compose/Theme/Typography/Typography/README.md) 表示材料设计类型规模。 Material Design
类型范围包括一系列对比风格，可支持您的产品及其内容的需求。 字体比例是字体系统支持的十三种样式的组合。 它包含可重复使用的文本类别，每个类别都有预期的应用和含义。
